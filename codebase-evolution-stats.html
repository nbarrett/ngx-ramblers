Key Insights

From One Group to a Platform

What started as East Kent's local website in 2013 is now a turnkey CMS that any Ramblers group can adopt without writing a line of code. The same codebase powers Kent Area, Sevenoaks, Berkshire and a growing list of groups - one volunteer effort, many sites.

13-Year Journey

From a humble AngularJS app in December 2013 to a modern 235K LOC TypeScript monorepo across three major repository transitions.

1,846 Commits

Consistent development: 750 in ekwg, 435 in ng-ekwg, 661 in ngx-ramblers. Peak years: 2014 (370) and 2026 (324 so far).

10x Code Growth

From ~21K LOC to 235K LOC. Growth has accelerated since July 2024 - the codebase has more than quadrupled (+305%) in under two years.

Server Transformation

Server grew from 6K to 78K lines (+1,167%) since July 2024. Complete JS to TypeScript migration with a Salesforce API layer and email composer.

Clean Code Philosophy

Only 217 comments across 235K TypeScript lines (0.1%). Self-documenting code with meaningful names.

Technology Evolution

Started on AngularJS 1.1 in 2013 and upgraded through every major version since - AngularJS 1.1 → Angular 7/14 → Angular 20. JavaScript → TypeScript. CSS → Sass. Express.js throughout.

Platform Baseline

As of March 2026 the project baseline is Node 24 LTS across local development, GitHub Actions, and Docker builds.
